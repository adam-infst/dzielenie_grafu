Kompilowanie javy:
javac GraphDrawer.java
(trzeba mieć pobrany java compiler, użyta wersja: 11.0.19)

Uruchamianie javy:
java GraphDrawer graph_to_matrix.txt
java GraphDrawer graph_to_matrix.txt groups.txt
(1szy plik zawiera macierz sąsiedztwa wraz z połączeniami, ewentualny 2gi plik wierzchołki podzielone na grupy)
(trzeba mieć pobraną java vm, użyta wersja: 11.0.19)
